# AutoPCR Android v1.0.1 发布说明

## 版本信息
- 版本号: v1.0.1
- 版本代码: 2
- 发布日期: 2026-02-10
- 文件大小: 74.6 MB

## 更新内容
- 修复了数据管理模块的若干问题
- 优化了数据库性能
- 添加了新的功能模块 (abyss.py, mirage.py)
- 改进了 HTTP 服务器功能
- 更新了用户界面布局
- 增强了错误处理机制

## 安装说明
1. 下载 APK 文件 `AutoPCR-v1.0.1.apk`
2. 在 Android 设备上启用"未知来源"安装
   - 设置 → 安全 → 未知来源 (允许安装非应用商店应用)
3. 点击 APK 文件进行安装

## 系统要求
- Android 5.0 (API 21) 或更高版本
- ARM64-v8a 或 armeabi-v7a 架构
- 建议 2GB 以上 RAM

## 功能特性
- 自动化公主连接游戏任务
- 支持多种游戏功能模块
- 内置 HTTP 服务器管理界面
- 数据库本地存储
- Python 脚本执行环境

## 注意事项
- 首次运行可能需要一些初始化时间
- 建议在稳定的网络环境下使用
- 请确保设备有足够的存储空间
- 使用前请仔细阅读使用说明

## 已知问题
- 某些设备上可能存在兼容性问题
